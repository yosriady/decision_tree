# DecisionTree

Decision Tree learning, implemented in Elixir.

### [API Documentation]()

## Installation

Add decision_tree to your list of dependencies in `mix.exs`:

```
    def deps do
      [{:decision_tree, "~> 0.0.1"}]
    end
```

## Running Tests

```
mix test
```

## Running Type Checker

> You need to have [dialyxir](https://github.com/jeremyjh/dialyxir) installed.

```
mix dialyzer
```
